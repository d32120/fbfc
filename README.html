FBFC - Francesco's BrainF**k Compiler

What is BrainF**k?

Brainf**k is a minimal esoteric programming language (esolang). It operates in an array of bytes of fixed size, with the index taken as its modulus calculated on the grid size It consists in only eight symbols,which performs different actions:

  • + : increment the value in the current cell
  • - : decrement the value in the current cell
  • < : moves the pointer to the previous cell (if the cell is the first, it goes to the last)
  • > : moves the pointer to the next cell (if the cell is the last, it goes to the first)
  • [ : starts a loop
  • ] : closes a loop. The code inside the loop is executed until the cell where the pointer is before the end of the loop is non-zero
  • . : prints the value inside the cell pointed by the pointer taken as an ASCII character
  • , : asks the user for an input. The input must be 1-character long and ASCII-representable (not something as ϝ)

All the remaining characters are considered a comment.

Caratteristiche

Nonostante la sua semplicità apparente, Brainfuck è Turing completo, il che significa che può teoricamente eseguire qualsiasi calcolo computabile.
